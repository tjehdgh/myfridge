# My Fridge
---
![main](https://github.com/tjehdgh/myfridge/assets/146055280/c215bc61-fd05-4447-9b8e-309c9b8410e0)
<br>
* 다들 어떠헌 요리를 한 이후에 남는 
재료들로 인해 곤란했던 경험이 있었을 것입니다.
이로 인해 지금 가지고 있는 재료로 
가능한 요리들을 찾을 수 있는 기능이 있었으면 하는 바람을 가졌습니다.
따라서 저희는 남는 재료로 가능한 
레시피를 찾아주는 저희 'My fridge' 프로젝트를 기획하게 되었습니다.
